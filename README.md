# ToxMCP Suite

**ToxMCP** is a suite of **guardrailed, auditable agentic workflows** for computational toxicology delivered through the **Model Context Protocol (MCP)**.

Preprint (bioRxiv): https://doi.org/10.64898/2026.02.06.703989

## Repositories

- **AOP MCP** — https://github.com/ToxMCP/aop-mcp
- **EPA CompTox MCP** — https://github.com/ToxMCP/comptox-mcp
- **ADMETlab MCP** — https://github.com/ToxMCP/admetlab-mcp
- **O-QT MCP (OECD QSAR Toolbox)** — https://github.com/ToxMCP/oqt-mcp
- **PBPK MCP** — https://github.com/ToxMCP/pbpk-mcp

## Notes

- Some modules depend on proprietary or rate-limited upstream services (e.g. OECD QSAR Toolbox, API keys). See each repo’s README.
- This umbrella repo is the entry point; each MCP server is versioned and released independently.
